# 📊 Aplicación Web de Control de Ventas y Caja

## 📝 Descripción
Esta aplicación web permite registrar y gestionar las **ventas diarias**, **gastos** y **entradas de dinero** de la empresa.  
El sistema divide el dinero en dos cajas: **Caja Principal** y **Caja Menos**, y facilita el **cuadre de caja diario**.  
Además, incluye una **gráfica mensual** para visualizar el total de ventas acumuladas.

## 🚀 Funcionalidades
- Registro de **ventas** con:
  - Número de factura
  - Método de pago: **Efectivo**, **Tarjeta**, **QR**
  - Asociación automática a Caja Principal o Caja Menos (si es efectivo)
- Registro de **gastos** diarios
- Registro de **entradas de dinero**
- **Cuadre de caja diario**: validación de ingresos vs egresos
- **Gráfica mensual de ventas** (usando Chart.js o Recharts)
- Interfaz sencilla y fácil de usar

## 🛠️ Tecnologías sugeridas
- **Frontend**: React
- **Backend**: Node.js + Express
- **Base de datos**: MongoDB o PostgreSQL
- **Gráficas**: Chart.js / Recharts
- **Control de versiones**: GitHub
